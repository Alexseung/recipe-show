## 프로젝트 설명

이 프로젝트는 기본적인 웹 페이지 레이아웃을 포함합니다. `header`와 `footer`는 화면이 렌더링될 때 자동으로 로드되며, 별도로 작성할 필요가 없습니다. 다음은 현재 구현된 사항과 앞으로 구현해야 할 사항들입니다.

### Header
- **구현 완료:**
  - 디자인
  - 메뉴 링크 이동

- **구현 필요:**
  - 검색 기능
  - 메뉴 하단의 이동 선 (menu-bottom-line)

### Footer
- **구현 필요:**
  - 전체 Footer 내용
